


## Build instructions.
### Build for WINDOWS:
    pyinstaller --onefile --add-data "templates\*.sh;templates" openClaw.py

### Build for OSX:
    `pyinstaller --onefile --add-data "templates\:templates" openClaw.py`

`python -m venv .venv`

`python openClawLocal.py`

### Open OpenClaw Web Portal
`http://localhost:2026`
